# Glossary

| Begriff         | Definition                                                                                 |
| :-------------- | :----------------------------------------------------------------------------------------- |
| **Agent Skill** | Eine wiederverwendbare Prompt-Datei, die einem Agenten eine Rolle zuweist.                 |
| **Snapshot**    | Eine Markdown-Datei, die den Zustand des Projekts zu einem bestimmten Zeitpunkt einfriert. |
